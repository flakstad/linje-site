How it works

1. Send transactional email

POST messages with project-scoped tokens and strict sender-domain enforcement.

2. Receive inbound as events

Provision inboxes and receive signed webhook payloads, with attachment URLs.

3. Operate via control-plane API

Guardrails, telemetry, and security actions live under admin API routes.

Strong API story

Everything in the product can be done by API. Dashboard is optional convenience.

  • Project-scoped and admin-scoped auth
  • Idempotency support for mutating admin routes
  • Machine-readable OpenAPI contract for `/admin/*`

What Linje is not

Included

  • Transactional outbound email
  • Inbound webhooks with retries
  • Operational control-plane APIs

Explicitly excluded

  • Campaigns and list management
  • Template editors and marketing automation
  • Open/click analytics suite
